# Soltty Logo - Banana AI Prompt

## Primary Prompt (Recommended)

```
A modern, minimalist logo for "Soltty" - a command-line time tracking tool.
Design features a sleek stopwatch or clock icon merged with terminal/CLI elements
like a command prompt cursor or brackets. Color scheme: deep teal/cyan (#00BCD4)
and charcoal gray (#2C3E50) on white background. The wordmark "soltty" in lowercase
using a modern monospace font like JetBrains Mono or Fira Code. Clean, geometric
shapes with subtle gradient. Professional developer tool aesthetic. Flat design,
vector style, suitable for GitHub profile and app icons. 1:1 ratio, high contrast,
scalable design.
```

## Usage Instructions

1. Choose one prompt based on desired style:
   - **Primary**: Balanced, professional
   - **Alternative 1**: Geometric, modern
   - **Alternative 2**: Friendly, approachable
   - **Alternative 3**: Classic terminal aesthetic
   - **Alternative 4**: Premium, sophisticated

2. Generate with Banana AI or similar tools:
   - DALL-E 3
   - Midjourney
   - Stable Diffusion
   - Leonardo AI

3. Refine with follow-up prompts:
   - "Make it more minimal"
   - "Increase contrast for better visibility at small sizes"
   - "Try variations with different color schemes"
   - "Create a monochrome version for dark/light modes"

## File Naming Convention

Save generated logos as:
- `soltty-logo-primary.png` (main color version)
- `soltty-logo-dark.png` (for dark backgrounds)
- `soltty-logo-light.png` (for light backgrounds)
- `soltty-icon.png` (square icon only, 512x512)
- `soltty-favicon.png` (16x16, 32x32, 64x64)
